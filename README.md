# Job Portal System

A full-stack web and mobile job portal application that connects job seekers and employers through an organized and user-friendly recruitment platform.

## Overview

The Job Portal System simplifies the recruitment process by allowing employers to publish job opportunities and helping applicants search, filter, save, and apply for jobs efficiently.

The platform supports three main roles:

* Job Seeker
* Employer
* Admin

Each role has dedicated dashboards and functionalities.

## Features

### Job Seeker

* Secure account registration and login
* Email verification
* Profile management
* CV upload
* Advanced job searching and filtering
* Job applications
* Cover letters
* Saved jobs
* Application tracking

### Employer

* Company profile management
* Post and manage job opportunities
* View applicants
* Review CVs and cover letters
* Update application status

### Admin

* Manage users
* Manage jobs
* Manage applications
* Monitor platform activity

## Smart Features

* AI skill suggestions
* Smart job matching
* Location suggestions
* Role-based access control
